# Amplify sandbox is required for local dev

This app reads `amplify_outputs.json` (backend config: Cognito, AppSync, etc.) at
the project root — see `src/components/ConfigureAmplifyClientSide.tsx` and
`src/utils/amplify-server-utils.ts`. That file is git-ignored and generated by
the Amplify sandbox; it does not exist until you deploy one, and `npm run dev`
/ `npm run build` will fail without it.

Before running `npm run dev`, start the sandbox in a separate terminal:

```bash
npm run sandbox
```

This runs `ampx sandbox --profile allowance` (the `allowance` AWS CLI profile
must be configured locally) and stays running, redeploying on backend changes
under `amplify/`. Leave it running alongside `npm run dev`. To tear the sandbox
down: `npm run sandbox:delete`.
