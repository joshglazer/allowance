This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app), using [AWS Amplify Gen 2](https://docs.amplify.aws/nextjs/) for auth and data.

## Amplify backend (required before running locally)

The app reads `amplify_outputs.json` at the project root for its Cognito/AppSync
config. That file is generated by the Amplify sandbox and is git-ignored, so it
won't exist on a fresh checkout — `npm run dev` and `npm run build` will fail
without it.

In a separate terminal, start the sandbox and leave it running:

```bash
npm run sandbox
```

This deploys a personal backend to AWS using the `allowance` CLI profile
(configure it with `aws configure --profile allowance` if you haven't) and
redeploys automatically as you edit files under `amplify/`. To delete the
sandbox stack when you're done: `npm run sandbox:delete`.

## Getting Started

With the sandbox running, start the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.
